How CSS is Important

CSS is a style sheet language that gives a website it's look, format and layout. Without CSS, a website will look bland, unappealing to the eyes and difficult to navigate. CSS is important because it controls all design-related aspects of a website, CSS controls the typography, colours, page layouts and every other visual aspects.

Difference between Paddings, Margin and Borders

Padding: Padding is used to create space around an element's content, inside of any defined borders. The padding property controls the space inside an element

Margin:A margin is the space around an element's border. The margin property controls the space outside an element.

Border:Every html element is a box shape element and has a border around them that can be set visible using the css border property to set the width, shape, color and style.
